# 🧙‍♂️ Workout Wizard

Workout Wizard is a fitness orchestration engine that uses Local LLM inference (Rust/llama.cpp) and Tiered Visuals (Jetpack Compose) to transform a standard training session into a loosely connected Dungeons and Dragon themed, wizardly experience.

## 🛠 Tech Stack

| Layer             | Technology                               | Purpose                                      |
| :---------------- | :--------------------------------------- | :------------------------------------------- |
| **Frontend** | Kotlin & Jetpack Compose                 | 120Hz Fluid UI & "Tier V" Visual Shaders    |
| **Intelligence** | Rust & `llama-cpp-2`                     | High-performance, private LLM inference       |
| **Local Storage** | SQLite (Room)                            | User history, workout logs, and session data |
| **Logic Engine** | Custom Rust Core                         | Tactical exercise selection & Tier 4 Oracle  |
| **Architecture** | Multiplatform (JNI/UniFFI)              | Seamless Rust <-> Kotlin communication       |
